# Enciclopédia de Câmera e Lente — Cinema Cadaxa

Linguagem de produção precisa. "Camera moves cinematically" não é direção. "Slow 50mm dolly-in, starting medium shot, ending close-up, over 8 seconds" é direção.

## 2. Movimentos de Câmera

| Movimento | Descrição | Efeito Dramático | Prompt |
|-----------|-----------|-----------------|--------|
| **Static camera** | Sem movimento | Observação, estabilidade, presença | "Static shot on tripod." |
| **Slow push-in / Dolly-in** | Câmera move fisicamente para perto | Intimidade crescente, tensão acumulando | "Slow dolly-in from medium to close-up." |
| **Fast push-in** | Dolly rápido | Urgência, choque, revelação | "Fast push-in, snap zoom feel." |
| **Pull-back / Dolly-out** | Câmera afasta do sujeito | Isolamento revelado, contexto expandido | "Dolly-out revealing the vast landscape." |
| **Tracking shot** | Paralela ao sujeito em movimento | Acompanhamento, energia, fluxo | "Lateral tracking shot following character." |
| **Pan left / Pan right** | Rotação horizontal sem mover câmera | Revelar, paisagem, transição | "Slow pan right across city skyline." |
| **Tilt up / Tilt down** | Rotação vertical | Revelar, poder, contemplação | "Tilt down from face to hands holding letter." |
| **Handheld micro-shake** | Shake sutil, realismo | Documentary, urgência, intimidade | "Handheld, slight micro-shake, documentary." |
| **Whip pan** | Pan extremamente rápido | Corte de energia, transição estilística | "Whip pan transition to next scene." |
| **Snap zoom** | Zoom rápido | Choque, comédia, ênfase dramática | "Snap zoom in to close-up at 3-second mark." |
| **Rack focus** | Foco muda de fundo para frente | Revelação, shift de atenção | "Rack focus from foreground cup to character." |
| **Orbit / 360** | Câmera orbita ao redor do sujeito | Exposição, heroísmo, produto | "Camera orbits 45 degrees around subject." |
| **Gimbal glide** | Movimento fluido, estabilizado | Polido, comercial, suave | "Smooth gimbal glide, fluid movement." |
| **Crane shot** | Arco vertical amplo | Escala épica, revelar ambiente | "Crane shot starting at ground, ascending." |
| **Aerial / Drone** | Vista de cima, movimento suave | Escala, estabelecer, grandiosidade | "Sweeping aerial drone shot over landscape." |
| **Hitchcock zoom** | Dolly-out + zoom simultâneo | Vertigem, desorientação psicológica | "Dolly-out while zooming in, Hitchcock effect." |

### Regra Universal de Movimento
Um movimento dominante por plano de 5 segundos. Adicione uma micro-ajuste sutil no máximo (handheld shake + dolly). Mais que isso produz caos visual.

## 3. Linguagem de Lente

| Lente | Characteristics | Quando Usar |
|-------|----------------|------------|
| **24mm** | Wide, imersivo, distorção de proximidade exagerada | Espaço imersivo, ação, environment grande |
| **35mm** | Natural, documentário, levemente wide | Documentary, drama cotidiano, conversas |
| **50mm** | Íntimo, perspectiva humana, neutro | Padrão para drama, natural para o olho |
| **85mm** | Retrato, fundo comprimido, flatters faces | Close-ups emocionais, retratos, performances |
| **100mm macro** | Textura, detalhe extremo, profundidade rasa | Inserts de objetos, detalhes de produto, micro-ações |
| **Anamórfico 40mm** | Widescreen cinematográfico, lens flares horizontais | Épico, sci-fi, alto cinema |
| **Shallow DOF** | Fundo desfocado, sujeito em foco | Foco emocional, bokeh, intimidade |
| **Deep focus** | Tudo em foco | Ambiente importante, pressure do espaço |
| **Compressed telephoto** | Fundo comprimido, foreground/background relação | Drama psicológico, espaço denso |

### Como Usar Lentes nos Prompts

Para Veo/Nano Banana — declare a lente diretamente:
```
"Medium close-up, 85mm, slow push-in."
"Static wide shot, 24mm, deep focus."
"Extreme close-up, 100mm macro, static."
```

Para Seedance — use junto com camera body:
```
"Sony FX3 handheld look. 35mm for medium shots, 85mm for close-ups, 100mm macro for inserts."
```

Para imagens (Nano Banana Pro) — use descritivo em vez de números:
```
"Shallow depth of field" em vez de "f/1.8"
"Portrait lens compression" em vez de "85mm"
"Wide immersive" em vez de "24mm"
```

## 8. Instruções de Câmera por Modelo

### Para Veo 3.1 / Nano Banana Pro
- Coloque instrução de câmera NO INÍCIO do prompt
- "Slow dolly-in shot." → depois o resto
- Especifique velocidade: "slow", "gradual", "quick"
- Especifique duração: "over 8 seconds"
- Use "no camera shake" ou "smooth gimbal" para limpar

### Para Seedance 2.0
- Declare o body de câmera para referência visual: "Sony FX3 handheld look"
- Use sintaxe de corte: "Shot 1. [...] Cut to. Shot 2. [...]"
- Combine lentes por função: "35mm para médios, 85mm para close-ups emocionais"
- Use `--camerafixed true` para câmera estática, `--camerafixed false` para movimento

### Para Geração de Imagem
- Use descritivo, não numérico: "shallow depth of field" não "f/1.8"
- Nano Banana ignora números de lente — use: "portrait lens compression", "wide immersive"
- GPT Image 2 aceita "50mm feel" como high-level look
