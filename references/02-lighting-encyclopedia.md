# Enciclopédia de Iluminação — Cinema Cadaxa

Use linguagem de produção precisa. "Cinematic" não é direção. "Warm window key from frame-left, cold rim backlight from frame-right, soft bounce fill from below" é direção.

---

## 1. Estruturas de Iluminação de Estúdio

### Three-Point Lighting (Setup Profissional Universal)
- **Key light:** Fonte principal, define sombras. 45° à frente-lateral do sujeito.
- **Fill light:** Preenche sombras do lado oposto. Mais suave que o key (50% da intensidade).
- **Rim / Back light:** Separa sujeito do fundo, cria profundidade. Vindo de trás-lateral.
- Uso: produtos, retratos, entrevistas, comerciais.
- Prompt: "Three-point softbox setup, warm key from frame-left, cool rim backlight from behind-right."

### Single Key Light
- Fonte única criando sombras dramáticas.
- "Single key light from upper left, creating dramatic shadow that cuts across the face."
- Uso: drama, noir, psicológico.

### Ring Light (Frontal Flat)
- Luz frontal uniforme, sem sombras visíveis, catch light circular nos olhos.
- "Ring light, flat frontal, beauty/fashion editorial style."
- Uso: beleza, moda, conteúdo digital, close-ups.

### Softbox Setup
- "Three-point softbox setup" — iluminação uniforme para produtos/retratos.
- "Single softbox, camera right" — dramático mas suave.

## 3. Direção de Luz

| Direção | Efeito | Uso |
|---------|--------|-----|
| Top light | Dramatiza rostos, olhos em sombra | Isolamento, julgamento, vilão |

| Backlight | Silhueta ou rim glow | Mistério, reveação, épico |
| Underlight | Inquietante, não-natural | Horror, alienação, comédia dramática |
| Frontal soft light | Plano, flatters | Beleza, comercial, acessível |
| Rim light | Separa do fundo, edge glow | Herói, poder, contorno dramático |
| Bounce fill | Suave, remove sombras duras | Fill natural, interiors |
| Window key light | Natural motivado, direcional | Realismo, intimidade, documentary |
| Motivated practical | Fonte visível no frame | Autenticidade, coerência de mundo |

## 5. Paletas de Cor Cinematográficas Nomeadas

Use paletas com cores concretas, nunca "cinematic colors":

### Fincher Cold
"Cold blue-gray shadows, desaturated skin tones, black negative space, no warm yellow tones."
- Referência: Seven, Fight Club, Gone Girl

### Deakins Natural
"Warm amber interiors, cool blue exteriors, clean contrast, natural motivated sources."
- Referência: No Country for Old Men, Blade Runner 2049

### Wong Kar-wai
"Saturated warm reds and deep greens, hazy practicals, nostalgic glow, tungsten spill."
- Referência: In the Mood for Love, Chungking Express

### Glazer Neon
"Pure black backgrounds, one dominant neon hue (magenta or teal), hard edges, no fill."
- Referência: Under the Skin

### Commercial Creamy
"Warm creams, soft pastels, clean whites, no harsh blacks, lifted shadows."
- Uso: anúncios de luxo, lifestyle, beleza

### Safdie Chaotic
"Mixed color temperatures, overlapping neon spill, urban contamination, no clean sources."
- Referência: Uncut Gems, Good Time

### Naturalistic Documentary
"Mixed practical sources, available light, slight color contamination, realistic skin."
- Uso: documentário, UGC, drama íntimo

### Golden Hour
"Warm amber-orange tones, long shadows, lens flare, backlit glow, romantic."
- Uso: épico, inspiracional, romance

### Blue Hour
"Cool twilight, deep blues, low contrast, ethereal, transitional."
- Uso: melancolia, transição, mistério suave

## 6. Estilos de Iluminação por Gênero

### Domestic Tragedy (Tragédia Doméstica)
```
Cold night interior, blue-green refrigerator light as key, minimal ambient fill,
no warm tones, harsh shadows, isolated pools of light in darkness.
```

### Noir
```
Chiaroscuro, strong side light from practical source, deep shadows,
venetian blind bars of light, high contrast, desaturated except for practicals.
```

### Commercial / Produto
```
Clean three-point setup, soft key from camera-left, subtle rim,
bounced fill, no harsh shadows, product clearly lit, background clean.
```

### Fashion Editorial
```
Single key, controlled backlight, rim light emphasizing silhouette,
precise shadows, high contrast but clean, motivated source visible.
```

### Documentary / UGC
```
Available light, practical sources only, slight color contamination acceptable,
natural skin tones, realistic exposure, slight underexposure preferred.
```

### Thriller / Psicológico
```
Low-key, single practical, deep shadows, obstructed frame, foreground dark,
subject partially lit, negative space threatening, cool palette.
```

### Epic / Adventure
```
Golden hour or blue hour, volumetric atmosphere, wide rim light,
environment as light source (sky, fire, water reflection), large scale.
```

## 7. Construção de Prompt de Iluminação

### Estrutura Completa
```
[Fonte principal] as [key/rim/fill], [direção] from [frame-left/frame-right/above/behind],
[qualidade], [temperatura de cor]. [Fonte secundária] as [fill/rim], [direção].
[Paleta geral com cores concretas].
```

### Exemplos

**Cena íntima de café:**
```
Soft afternoon window light from frame-left as warm key, creating gentle rim on hair.
Practical table lamp from frame-right as warm fill. Warm amber-cream palette,
no cold tones. Shallow depth of field, bokeh background.
```

**Cena de escritório de madrugada:**
```
Cold fluorescent overhead as key, harsh top light, crushed shadows under eyes.
Monitor glow as secondary fill, cold blue-white from front-below.
Cold blue-gray palette, desaturated skin, black negative space.
```

**Cena de perseguição noturna:**
```
Sodium vapor streetlamps as motivated backlight, warm amber rim.
Neon spill from storefronts as fill, mixed color contamination.
Wet pavement reflections, urban night palette, Safdie chaotic style.
```
